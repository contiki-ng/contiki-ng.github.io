int8_t ctr_crypt_check_status ( void  )

Checks the status of the CTR crypto operation.

Returns
CRYPTO_PENDING if operation still pending, CRYPTO_SUCCESS if successful, or CRYPTO/AES/CTR error code
Note
This function must be called only after ctr_crypt_start().
