int8_t cbc_crypt_check_status ( void  )

Checks the status of the CBC crypto operation.

Returns
CRYPTO_PENDING if operation still pending, CRYPTO_SUCCESS if successful, or CRYPTO/AES/CBC error code
Note
This function must be called only after cbc_crypt_start().
