#define NRF_TWIS_PATCH_FOR_MPW3   1

Definition for patching PAN problems.

Set this definition to nonzero value to patch anomalies from MPW3 - first lunch microcontroller.

Concerns:

  • PAN-29: TWIS: incorrect bits in ERRORSRC
  • PAN-30: TWIS: STOP task does not work as expected
