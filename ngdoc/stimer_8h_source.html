45 /**
46  * \defgroup stimer Seconds timer library
47  *
48  * The stimer library provides functions for setting, resetting and
49  * restarting timers, and for checking if a timer has expired. An
50  * application must "manually" check if its timers have expired; this
51  * is not done automatically.
52  *
53  * A timer is declared as a \c struct \c stimer and all access to the
54  * timer is made by a pointer to the declared timer.
55  *
56  * \note The stimer library is not able to post events when a timer
57  * expires. The \ref etimer "Event timers" should be used for this
58  * purpose.
59  *
60  * \note The stimer library uses the \ref clock "Clock library" to
61  * measure time. Intervals should be specified in the seconds.
62  *
63  * \sa \ref etimer "Event timers"
64  *
65  * @{
66  */
